This is my Kitchen Mod. It has little functionality so far but still try it. A watch out for the taps. They are rubbish. 

#wxSand: Owen Piette's Falling Sand Game - The Cooking and Misc Mod
#Version 4.3, File subversion 2
#element element r g b gravity slip density conductivity visible

group Cooking Hand Metal HeatRing Burner Tap Water
group Food/Drink Water Meat Vegetables

#The deep, dark void.
element Empty 0 0 0 0 0 1 0 1

#Your hand. Use to turn tap On/Off. This can only be 1 pixel or it will NOT work!
element Hand 255 0 200
neighbor 1.0 Hand Empty 1.0 Empty Empty
neighbor 1.0 Hand Metal 1.0 Metal Metal
neighbor 1.0 Hand HeatRing 1.0 HeatRing HeatRing
neighbor 1.0 Hand Burner 1.0 Burner Burner

#Heats up. Use in an Oven/Cooker, Kitchen Sink and Kettle
element Metal 200 200 200 0 0 1 .1 1
hotcolor Metal 255 50 0

#A ring of metal to be heated up for boiling water. For use in a Kettle.
element HeatRing 100 100 100 0 0 1 .5 1
hotcolor HeatRing 255 50 0
self 1.0 HeatRing 1.0 HeatRingOn
hotself 50 HeatRing
element HeatRingOn 255 100 100 0 0 1 1.5 0

#Heat up to turn on. Lets off fire. For use in an Oven/Cooker.
element Burner 149 149 149 0 0 1 .5 1
hotcolor Burner 255 50 0
element BurnerOn 149 149 149 0 0 1 .5 0
hotcolor BurnerOn 255 50 0
self 1.0 Burner 1.0 BurnerOn
hotself 90 Burner
neighbor 1.0 BurnerOn Empty 1.0 BurnerOn Fire
neighbor 1.0 BurnerOn Burner 1.0 BurnerOn BurnerOn

#A water tap. Use hand to turn it on and off. You have to wait for a while after turning it off before turning it on again.
element Tap 137 137 137 0 0 1 0 1
element TO 137 137 137 0 0 1 0 0
element TOF 137 137 137 0 0 1 0 0
element TC 137 137 137 0 0 1 0 0
neighbor 1.0 Hand Tap 1.0 Tap TO
neighbor 1.0 TO Tap 1.0 TO TO
neighbor 1.0 Hand TO 1.0 TOF TOF
neighbor 1.0 TOF TO 1.0 TOF TOF
neighbor 1.0 TOF Tap 1.0 TOF TOF
neighbor 1.0 TC TOF 1.0 TC TC
neighbor 0.01 TO Empty 1.0 TO Water
neighbor 1.0 TC TO 1.0 TC TC
self 0.0001 TOF 1.0 TC
self 0.005 TC 1.0 Tap

#A nice refreshing drink.
element Water 50 50 200 .9 1 1 .75 1
element Steam 240 240 255 -1 1 0 1.01 0
hotcolor Water 240 240 255
self 0.1 Water 1.0 Steam
hotself 90 Water

#Fire, Burns/Heats up Things. A source of energy.
element Fire 255 255 0 -1 1 1 1.1 1
hotcolor Fire 255 0 0
self 0.1 Fire 1.0 Empty

#Minced meat. Not suitable for vegetarians.
element Meat 255 160 200 .95 0 1 .5 1
element CookedMeat 150 0 70 .95 0 1 .5 0
hotcolor Meat 150 0 70
hotcolor CookedMeat 125 0 50
neighbor 0.075 CookedMeat Meat 1.0 CookedMeat CookedMeat
self 0.75 Meat 1.0 CookedMeat
hotself 90 Meat

#Sliced and Diced Vegetables. Suitable for vegetarians.
element Vegetables 0 200 0 .9 1 1 0 1
element Lettuce 0 200 0 .9 1 1 0 0
element Carrot 255 100 0 .9 1 1 0 0
element Cauliflower 255 255 255 .9 1 1 0 0
element SoupVeg 255 50 0 .9 1 1 .5 0
element Soup 255 150 0 .9 1 1 .5 0
self 1.0 Vegetables 0.3 Lettuce 0.3 Cauliflower 0.4 Carrot
neighbor 1.0 Lettuce Water 1.0 SoupVeg Soup
neighbor 0.01 Soup Water 1.0 Soup Soup
neighbor 0.01 SoupVeg Lettuce 1.0 SoupVeg SoupVeg
neighbor 0.01 Carrot SoupVeg 1.0 SoupVeg SoupVeg
neighbor 0.01 Cauliflower SoupVeg 1.0 SoupVeg SoupVeg
neighbor 0.1 Soup SoupVeg 1.0 SoupVeg Soup
neighbor 0.01 Soup Water 1.0 Soup Soup

#Piecrust. Make a bowl of this, pour in some meat, add a pie top and cook.
element Piecrust 220 190 150 0 0 1 .5 1
element CCrust 150 100 50 0 0 1 0 0
hotcolor Piecrust 150 100 50
self 0.1 Piecrust 1.0 CCrust
hotself 90 Piecrust
neighbor 0.01 CCrust Piecrust 1.0 CCrust CCrust
