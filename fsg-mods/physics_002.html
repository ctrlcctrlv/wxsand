Purple100 here!

This mod that I created has very good organization - each element is placed in a group with similar things. Also, all falling elements have spouts. Very useful if you need to start a mod but don't want to spend time making the spouts and organizing elements. Includes Tar (from Lynx's recent mod "Elements Galore" - I highly recommend it!) and Pitch, the solid variant of Tar. Salt's behavior in water has been tweaked to be more realistic and better for making Saltwater - no more expanding liquids as a result of adding it. Try it out.
