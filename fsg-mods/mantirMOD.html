Version 5 of LordMantir's main modding project. The second major rebuild - other versions were relatively minor updates and conversions so as to make it work with new versions of wxSand. Now only works with wxSand v4.3 and above, so don't try it with purple's version ad complain that there's an error. One main thing I've done is make a working 'wave' system, wherby dropping rubble onto water creates a wave that makes spray and travels through the water. Also, dropping lots of water onto a tray of wall has a cool effect.

I've tried to make sure everyhting does something - any expected reactions should be in there. This is relatively easy at the moment, but as it increases in elements I'll try to keep it that way. I've also made sure everything looks good - fire is a good example. 

#wxSand: Owen Piette's Falling Sand Game
#Version 4.4, File subversion 2

#element element r g b gravity slip density conductivity visible
#self probability element [prob. result] [...]
#hotself temp element
#hotcolor element r g b
#neighbor probability element1 element2 [prob. result1 result2] [...]
#hotneighbor temp element1 element2

#explosion neighbor1 neighbor2 xspeed<255 yspeed<255
#pexplosion neighbor1 neighbor2 diameter element

#Liquids = 0.21 to 0.5
