Purple100 here!

This is a glider mod that I built some time ago. Put the glider together in a 2x3 formation with the game paused, and when it is unpaused, the glider will move in a straight line (either horizontally or vertically) depending on which direction you made the front point.

#Name:  Glider Mod
#Author:  Purple100
